## Step 1: Create a new folder
I created a new folder on my computer for my project using:
```
C:\>mkdir newrepo
C:\>cd\newrepo
```
## Step 2: Initialize Git repository
I initialized a new Git repository inside the folder using:
`C:\newrepo>git init`
## Step 3: Create README file
`C:\newrepo>echo. > readme.md`
## Step 4: Add files to staging area
I added all files to Git using:
`C:\newrepo>git add readme.md`
## Step 5: Commit changes
I saved the changes with a commit message:
`C:\newrepo>git commit -m"file created"`
## Step 6: Connect to GitHub repository
I linked my local repository to GitHub using
`C:\newrepo>git remote add origin git@github.com:hanaghonim609-jpg/howtocreatnewrepo.git`
## Step 7: Push to GitHub
Finally, I pushed the project to GitHub using:
`C:\newrepo>git push -u origin master`
